付加説明：今回はifをより深く入り、elifとelseを活用してみる回であった。最初のいくつかはifをelifにそのまま変えても余り変わっていることがなく、
ifを重複で使うことが多かった。しかし、アカウントとパスワードを入力するところでif重複のエラーが生じた。
아이디 = input("アカウント入力 :")
if 아이디 == "admin":
    print("hello, admin :)")
    if 패스워드 == "admin":
        print("login success :)")
    else :
        print("login fail :(")
else :
    print("not user :(")
これはアカウントとパスワード入力するとき成功と失敗を表せるように作られたプログラムだが、最初、自分が組んだコードは
아이디 = input("アカウント入力 :")
if 아이디 == "admin":
    print("hello, admin :)")
else :    
    print("not user :(")
if 패스워드 == "admin":
        print("login success :)")
else :
    print("login fail :(")
であり、このコードならアカウントを打ち間違っても、途中に止まることなくそのまま実行される。そして途中にこのようなエラーが出る
name '패스워드' is not defined
これを解決するため、アカウント入力の成功の場合、失敗の場合、そして、成功したらパスワードの成功の場合と失敗の場合を考えてコードを組んでみた。
その結果が『アカウント入力に成功したら』という1つ目のifに2つ目のif「パスワード成功の場合」が従属されることがわかった。
さらにカフェのキオスクプログラム問題では、elseにもifがつき,
if a ==1:
    if b==1:
        print("アイスアメリカーノ")
    else:
        print("ホットアメリカーノ")
else :
    if b==1:
        print("アイスカフェラテ")
    else:
        print("ホットカフェラテ")
このようにelseにもifをつける事でできるとわかった。ifを使うときに一番大事なのはifで仮定した条件通りにいかなかった場合を考えておく力を身につけることが大事だとわかり
次回はif文で習得したことを応用して多様なプログラムを組んでいく。
