一番エラーが生じた問題は『n分前の時間』であり、かなり詰まってしまったところであった。
最初に
h = int(input("何時なのか入力(午後、午前分け不可) : "))
if h >=24 or h<0 :#ここなんだが、時間を表わすプログラムは２４時間以前まで表現しているからこの条件を入れた。この条件か成り立つと
print("入力エラー")と出力される。そしてそのままプログラムが終了されるようにし設計した。
elseを利用して正しく入力した場合を設定した。
時間は２４時間まで表せるように、分も同じく６０分以上は出力されないから同じく
  m = int(input("何分なのか入力　: "))
    if m>=60 or m<0:
        print("入力エラー"）
の条件も最初のifに従属した。時間と分を全部入力したからそれらを全部分に変え、何分前だから希望する分差し、それらをまた時間と分に表したら終わりだと
思ったがここで一番難しかったエラーが発生した。それは23:59時から00:00時になるところが25時とか26時に出力されたのだ。
このエラーも同じく条件付きで2つの場合を考える必要があった
        n = int(input("何分前なのか入力 : "))
        c = (60*h+m)-n
        if h==0 and m<n :
            print(n,"分前の時間は",(1440+c)//60%24,"時",(1440+c)%60,"分")
        else:
            print(n,"分前の時間は",c//60,"時",c%60,"分")
こっちらのコードの％24することで24時はなれないようにし、ifの条件が前のエラーを解決し、elseから他のところがうまく起動するようにしたのだ。
